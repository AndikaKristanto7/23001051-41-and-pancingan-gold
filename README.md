# 23001051-41-and-pancingan-gold

# Run Backend
1. Create postgresql database named "pancingan" (without quotes)
2. Go to ./backend
3. Run "npm install"
4. Run "knex migrate:latest"
5. Run "knex seed:run"
6. Run "npm start"
7. BE is up, ready to use 

# Run Frontend
1. Go to ./frontend
2. Run "npm install"
3. Run "npm run serve"
4. FE is up, ready to use
